# Root no password
A lot of times there is no password on the root account on #mysql, so you can just connect with: `mysql -u root`. Then you can use:

`show databases`

```
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| pentesterlab       |
| performance_schema |
+--------------------+
```

`use database`

`show tables`

`select * from table*`
